COStream

高性能
数据流编程语言

特性介绍 起步 GITHUB

高效

充分利用流程序的并行性
自动实现特定域中流应用的优化
提高程序员在数据流程序编程效率

直观

采用数据流图的方式来描述应用的处理过程，用节点表示计算，边表示数据的流动

多用途

胜任多项大数据量处理应用
如媒体处理、信号处理、搜索应用、数据文件处理
